Guide des Réseaux Informatiques

Tout comprendre sur les réseaux informatiques

Qu'est-ce qu'un réseau ?

C'est ici que nous mettrons l'introduction. Un réseau informatique est un ensemble d'équipements reliés entre eux pour échanger des informations. On peut parler des ordinateurs, des serveurs, des téléphones...

Les Types de Réseaux

Les réseaux sont classés selon leur étendue géographique. Voici les principaux types :

Type Signification Distance / Étendue Technologies Courantes
PAN Personal Area Network (Réseau Personnel) Quelques mètres (une pièce) Bluetooth, USB, NFC, RFID, IR
LAN Local Area Network (Réseau Local) 10m à 1km (bâtiment, maison) Ethernet (100Base-T, 1000Base-T), Wi-Fi, CPL
MAN Metropolitan Area Network (Réseau Métropolitain) Jusqu'à 100km (une ville) Fibre optique, Boucle locale radio, WiMax
WAN Wide Area Network (Réseau Étendu) Plus de 100km (pays, monde) Internet (IP), MPLS, Fibre optique, Satellite, Réseaux mobiles

Attention à ne pas confondre !

Ne confondez pas WAN (Wide Area Network, réseau étendu) et WLAN (Wireless Local Area Network, réseau local sans fil), qui est le terme technique pour désigner le Wi-Fi.

Les Adresses IP

Une adresse IP (pour Internet Protocol) est un numéro d'identification unique attribué à chaque appareil connecté à un réseau.

On l'appelle adresse logique car elle est assignée par le réseau. Elle sert à identifier l'appareil et indiquer son emplacement, un peu comme une adresse postale.

IP Statique vs. IP Dynamique

Une adresse IP peut être assignée de deux manières :

  • Adresse IP Statique (Fixe/permanente) : L'adresse est configurée manuellement et ne change jamais. Essentiel pour les serveurs ou les imprimantes réseau.
  • Adresse IP Dynamique (Temporaire) : L'adresse est attribuée automatiquement par un service DHCP lorsqu'un appareil se connecte. C'est le cas le plus courant pour nos PC et smartphones.

Les Adresses MAC

Une adresse MAC (pour Media Access Control) est un identifiant physique unique au monde, gravé sur la carte réseau d'un appareil par le fabricant.

On l'appelle adresse physique ou adresse Ethernet. Contrairement à l'adresse IP (logique), l'adresse MAC est permanente.

Son rôle principal est de permettre la communication entre les appareils au sein d'un même réseau local (LAN). Le routeur l'utilise pour savoir exactement à quel appareil envoyer les données sur le Wi-Fi.

Les Protocoles de Communication

Un protocole de communication est un ensemble de règles et de normes que les appareils doivent suivre pour échanger des données de manière fiable et ordonnée.

Ces règles définissent tout : le format des messages, l'identification, la détection et correction d'erreurs, etc.

Des exemples connus incluent TCP (Transmission Control Protocol) et UDP (User Datagram Protocol), qui travaillent souvent avec IP (TCP/IP).

Modes de Communication

La manière dont les données sont envoyées d'un expéditeur à un ou plusieurs destinataires.

Mode Principe Exemple d'utilisation
Unicast De un vers un. (1 expéditeur, 1 destinataire) Consultation d'une page web, envoi d'un e-mail.
Multicast De un vers un groupe. (1 expéditeur, plusieurs destinataires intéressés) Streaming vidéo en direct (ex: flux TV sur IP).
Diffusion (Broadcast) De un vers tous. (1 expéditeur, tous les destinataires du réseau) Demande DHCP (qui est sur le réseau ?). Les anciens Hubs fonctionnaient ainsi (obsolète).
